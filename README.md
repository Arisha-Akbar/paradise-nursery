# Paradise Nursery: Online Plant Store

A beautiful, responsive e-commerce frontend for browsing and purchasing houseplants, built with React, Redux, and Bootstrap.

ðŸ”— **Live Demo**: (https://paradise-nursery-react.netlify.app/)

## Features
- Browse 6+ houseplants across 3 categories
- Add/remove items from cart
- Update quantities in real time
- Fully responsive design
- Clean, modern UI with Bootstrap styling

## How to Run Locally
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000`

## Built With
- React
- Redux Toolkit
- React Router
- Bootstrap 5
